# SGA-Sistema-Gestao-Academico
Sistema de Gestão Acadêmica - Ciclo de vida de um sistema de informação de uma empresa

Sobre o Projeto - Este projeto consiste no desenvolvimento de um protótipo de alta fidelidade para um Sistema de Gestão Acadêmica (SGA), focado na digitalização e otimização dos processos administrativos da Secretaria e Diretoria de Ensino (Diren). O objetivo principal é a migração de processos manuais para um ambiente digital centralizado e seguro. 

Arquitetura de Segurança - O diferencial técnico deste projeto é a implementação nativa de conceitos avançados de Segurança da Informação:
    Tríade CID: O sistema foi projetado garantindo Confidencialidade, Integridade e Disponibilidade dos dados sensíveis. 
    Modelo Zero Trust: Diferente de sistemas convencionais, este SGA adota uma política de "Confiança Zero", eliminando o auto-registro de usuários e links públicos de recuperação de senha para mitigar ataques de engenharia social.  
    Provisionamento Centralizado: O cadastro de operadores é realizado exclusivamente pelo administrador do sistema.  ⚙️ Especificações TécnicasModelo de Processo: Cascata (Waterfall), garantindo um fluxo linear desde o levantamento de requisitos até a prototipagem.  
    
    Engenharia de Requisitos: Implementação completa de operações CRUD (Create, Read, Update, Delete) para gestão de discentes.  Resiliência: Planejamento de políticas de backup integral sazonais (a cada seis meses) para preservação do histórico escolar.  
    
    Ferramentas Utilizadas - Figma: Design de interface de alta fidelidade.  Draw.io: Modelagem lógica e fluxogramas de processos.  IA Assistiva (Gemini 3.1 Pro): Aplicação de Engenharia de Prompt para criação de layouts robustos e funcionais.  
    
    Visualização do Protótipo - O protótipo simula a experiência real de uso, incluindo telas de login administrativo, dashboard de indicadores e formulários técnicos de cadastro.
